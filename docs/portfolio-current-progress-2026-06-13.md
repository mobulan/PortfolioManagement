## 已交付能力

- 多组合、永久组合、全天候和自定义配置，支持归档及汇总是否包含归档组合。
- 同一资产类别多基金/ETF/现金/手工资产，实时估值复用和详细收益、占比、偏离展示。
- 买入、卖出、现金流、手续费、现金分红、红利再投、基金转换、手动调整。
- 交易计划/确认状态、交易编辑删除、影响预览、基线账本重放和本金记录。
- 按比例、填坑、削峰、强制再平衡、智能买卖草稿和实际成交批量回填。
- 手动/自动每日快照、同日覆盖确认、覆盖版本保留与恢复。
- JSON、CSV、XLSX/XLSM 只读导入；冲突预览、跳过/覆盖/合并和错误报告。
- 旧持仓与分组持仓迁移，CSV 空数据导出可直接作为标准模板。
- 总资产/本金/收益、资产配置、收益率、回撤、类别收益、交易节点和收益热力日历。
- 候选基金搜索、权重校验、定期/阈值再平衡、基金基准和真实历史净值回测。
- 年化收益、波动率、夏普、最大回撤、Calmar、胜率、最长回撤、Alpha/Beta、相关性、收益/风险贡献、20/60/120 日滚动波动率。
- 可命名、保存并下载独立回测报告。
- 数据质量检测、组合健康评分和带原因、动作的可解释配置建议。
- 与基估宝一致的深色玻璃卡片、青色主色、红涨绿跌，以及桌面/移动端响应式布局。

## 验证结果

- Node smoke：组合核心、导入、迁移、Dashboard、快照、交易全部通过。
- 生产构建：`npm run build` 通过。
- ESLint：零错误；仓库仍有 28 条投资组合范围外的既有 React Hook 警告。
- 浏览器点击回归：总览、交易、基金转换、历史、回测均已实际打开验证。
- 移动端：Firefox 390 x 844 实测，文档宽度与视口同为 390px，无页面级横向溢出。
- 空组合语义：不会生成 0 元买卖建议或把未建仓误报为四类资产低配。
- 开发控制台：产品运行路径零错误；异常浏览器语言值不会再使 React Query Devtools 抛错。

桌面端与移动端截图已用于运行时验收；本地 Playwright 产物在验收后按缓存清理规则删除，不纳入仓库。

## 尚需外部环境验证
