- 总资产随日期变化：对应 P0 “总资产走势”。
- 各项资产折线图：对应 P1 “资产类别收益率曲线”或“资产类别市值趋势”。
- 百分比堆叠图：对应 P1 “资产类别市值堆叠图 / 历史占比”。

## 2026-05-15 至 2026-05-16 新增完成项

### 1. 新增持仓金额处理

- 新增组合数字解析能力，支持 `39,202.20` 这类带千分位逗号的金额。
- 对非法格式提供用户可见错误提示，不再静默归零。
- 基金在按市值/本金模式录入时，会结合当前净值或估算净值自动换算份额。
- 补充 smoke test 覆盖金额解析、非法格式、按净值换算份额。

### 2. 持仓录入 UI

- 新增持仓里的资产类别从下拉框改为按钮组。
- 录入模式保持“按市值录入 / 按份额录入”分段按钮。
- 金额输入 placeholder 增加示例，减少格式误用。

### 3. 迁移面板

- 旧持仓迁移和分组持仓迁移面板完成中文化。
- 无可迁移记录时改为“暂无可迁移”状态提示，不再展示灰色禁用按钮造成“功能未实现”的误解。
- 迁移预览继续保持幂等：已有组合持仓不会重复导入。

### 4. CSV 与历史导入

- CSV 导入/导出面板完成主要中文化。
- CSV 冲突模式文案改为中文：跳过已有记录、覆盖已有记录、合并字段。
- CSV 数字字段复用统一数字解析，导入金额支持千分位格式。
