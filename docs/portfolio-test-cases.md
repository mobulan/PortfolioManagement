# Portfolio Test Cases

## Scope

Agent I validates portfolio behavior through the Node smoke script and documents manual follow-up cases for UI or integration paths that are outside this agent's write scope.

Automated script:

```bash
node scripts/portfolio-smoke-test.mjs
node scripts/portfolio-import-smoke-test.mjs
node scripts/portfolio-transaction-smoke-test.mjs
node scripts/portfolio-dashboard-smoke-test.mjs
node scripts/portfolio-migration-smoke-test.mjs
node scripts/portfolio-snapshot-smoke-test.mjs
```

## Current Residual Risks

- Invalid import relationships are filtered for data integrity; the UI reports dropped counts and errors, but users should still review previews before applying.
- Legacy and group migration previews are automated at the pure-helper layer and wired into the workspace; browser-level migration coverage is still pending.
- The automated smoke suite validates calculation, import, migration, snapshot, backtest, insights, and ledger modules. Supabase multi-device behavior still needs a valid external integration environment.
- Full-repository lint currently has 28 pre-existing React Hook warnings outside the portfolio module and no errors.
